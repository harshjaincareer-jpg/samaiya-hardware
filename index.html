🔒 One-time Firebase Setup

Do this once. After setup, data saves permanently in cloud and syncs on all phones automatically.

Step 1: Open console.firebase.google.com on a laptop/PC (ask someone or use a cyber cafe once)
Step 2: Click "Add project" → Name it samaiya-hardware → Continue → Create project
Step 3: Click "Web" icon (</>) → Register app → Copy the firebaseConfig object
Step 4: Go to Build → Realtime Database → Create database → Start in test mode → Enable
Step 5: Paste your config values in the fields below and tap Save

📱 Share with 2nd Person

1. Send the Samaiya_Hardware.html file to the second phone (via WhatsApp)
2. Open it in Chrome on the second phone
3. Go to Setup tab and enter the same Firebase config
4. Both phones will now sync automatically 🔁
